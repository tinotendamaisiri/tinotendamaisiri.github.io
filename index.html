MediCare Hub - Highly Available AWS Architecture (Reliability Pillar)
Description

Designed and deployed a highly available medical web application on AWS, re-architected from a single-instance setup into a fault-tolerant, multi-AZ system aligned with the Reliability Pillar of the AWS Well-Architected Framework.

Data Science/Engineering
SMOTE–WCGAN-GP Hybrid for Tabular Data Augmentation
Machine Learning • Data Augmentation • Research

Data Science • Machine Learning • Research

Conducted an in-depth academic study on tabular data augmentation methods for employee attrition prediction, analyzing both classical oversampling and generative approaches.

The project evaluates SMOTE, GAN-based models, and Wasserstein Conditional GAN with Gradient Penalty (WCGAN-GP), concluding that WCGAN-GP best preserves feature correlations, statistical distributions, and data privacy. A hybrid SMOTE–WCGAN-GP approach is proposed to address data sparsity and class imbalance.
